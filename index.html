O que é HTML?


HTML é uma linguagem de marcação que define a estrutura do seu conteúdo. HTML consiste em uma série de elementos que você usa para mostrar algo de uma determinada maneira ou agir de uma certo modo. As tags podem criar um hiperlink de uma palavra ou imagem para outro lugar, podem colocar palavras em itálico, podem aumentar ou diminuir a fonte e assim por diante.

O que é CSS?


Cascading Style Sheets (CSS) é uma linguagem usada para descrever a apresentação de um documento escrito em uma linguagem de marcação como HTML ou XML. CSS pode ser usado para estilos de texto de documentos muito básicos — por exemplo, para alterar a cor e o tamanho de títulos e links. Ele pode ser usado para criar um layout — por exemplo, transformar uma única coluna de texto em um layout com uma área de conteúdo principal e uma barra lateral para informações relacionadas. Pode até ser usado para efeitos como animações.

O que é JavaScript?


JavaScript é a linguagem de programação mais popular do mundo e é uma das principais tecnologias da World Wide Web, juntamente com HTML e CSS. Ela possui tipagem dinâmica, orientação a objetos baseada em protótipos e funções de primeira classe. Ela é multi-paradigma e suporta estilos de programação orientados a eventos, funcionais e imperativos.

O que é SEO?


SEO significa otimização para motores de busca e que diz respeito as estratégias usadas para ranquear um site dentro de mecanismos de busca como o Yahoo, o Bing e, claro, o mais famoso de todos, o Google.

O que é Design Responsivo?


Design responsivo é uma abordagem de desenvolvimento web que ajusta a aparência e o layout de um site ou aplicativo para diferentes tamanhos de tela e dispositivos (como smartphones, tablets e desktops). Isso é feito usando técnicas como media queries, layouts fluídos, e elementos flexíveis, garantindo boa usabilidade e visual consistente em qualquer plataforma.

Desafio HTML


Crie uma página HTML simples com uma estrutura básica contendo títulos (h1, h2, h3) para organizar seções como "Sobre Mim" e "Projetos". Adicione um parágrafo descritivo sobre você ou o site e exiba uma imagem ilustrativa. Insira um link para um site externo, como seu GitHub ou LinkedIn.

Inclua um formulário com campos para nome, e-mail, mensagem e um botão de envio. Crie uma lista (ordenada ou não) com suas habilidades ou hobbies, além de um menu suspenso (dropdown) para seleção de área de interesse. Adicione uma tabela com dados simples, como projetos ou cronograma.

Conecte a página a um arquivo styles.css para estilização, aplicando IDs e classes para facilitar o design.

Desafio CSS


Crie uma landing page de portfólio pessoal que utilize diversos conceitos fundamentais de CSS. A estrutura deve ser organizada com margin e padding, garantindo espaçamento adequado entre os elementos e dentro dos containers. Defina tamanhos específicos usando width e height, ajustando imagens, botões e seções conforme necessário. Experimente diferentes tipos de posicionamento como static, relative e absolute, destacando elementos como botões de ação.

A organização dos elementos deve utilizar propriedades de display, como block, inline e inline-block, para alinhar texto e imagens. Inclua uma imagem de perfil posicionada ao lado do texto na introdução, utilizando boas práticas de alinhamento horizontal e vertical. Personalize o design com estilos de fontes, aplicando tamanhos e tipos adequados para títulos, subtítulos e parágrafos.

Explore unidades de medida como px, %, em e rem para obter um design flexível e responsivo. Estruture o código com IDs para elementos únicos e classes para reaproveitar estilos. Adicione interatividade com efeitos visuais, como o hover, para alterar cores ou dimensões de links e botões ao passar o mouse.

Utilize box-sizing para ajustar o comportamento de largura e altura, e aplique Flexbox e Grid para organizar seções como a galeria de projetos, garantindo um layout dinâmico e bem estruturado.
